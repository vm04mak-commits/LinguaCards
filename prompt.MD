ПРОЕКТ: Telegram Mini App для изучения английского языка
Рабочее название

LinguaCards
(название не принципиально — архитектура и логика важнее)

1. Суть продукта

LinguaCards — Telegram Mini App, который позволяет:

учить английские слова через двусторонние карточки (RU ⇄ EN);

использовать готовые тематические наборы;

создавать свои карточки;

получать автоматический перевод при создании карточки;

обучаться прямо в Telegram без установки приложений;

при необходимости — платить через Telegram Stars.

2. Целевая аудитория

активные пользователи Telegram;

люди, которым нужен практический английский;

мигранты, путешественники, IT-специалисты, студенты;

пользователи, которые не хотят «курсы», а хотят слова по темам.

3. Ценностное предложение

нулевой порог входа (нет регистрации);

обучение в привычной среде Telegram;

карточки — быстрый и проверенный формат;

возможность учить свои слова;

мягкая, неагрессивная монетизация.

4. Функциональность продукта
4.1 Авторизация

через Telegram WebApp;

пользователь идентифицируется по Telegram ID;

без логинов, паролей, email.

4.2 Карточки

двусторонние:

лицевая сторона — слово на русском;

обратная — перевод на английский;

flip по нажатию;

кнопки:

«Знаю»

«Повторить».

4.3 Наборы карточек (Decks)
Типы:

Публичные (системные)

Кафе

Путешествия

Работа

Повседневный английский

Пользовательские

создаются пользователем;

приватные;

ограничены в бесплатной версии.

4.4 Режим обучения

показ карточек по одной;

приоритет:

новые;

в процессе;

изученные (редко);

сохранение прогресса.

4.5 Создание карточки с переводчиком

Пользователь:

вводит слово/фразу на русском;

нажимает «Перевести»;

приложение предлагает перевод;

пользователь:

принимает;

редактирует;

или заменяет.

Перевод не сохраняется автоматически — контроль остаётся у пользователя.

5. Переводчик (ключевая premium-функция)
Архитектура
Mini App → Backend → Translation API → Backend → Mini App

Реализация

перевод выполняется только на backend;

используется:

OpenAI / LLM (рекомендуется);

либо Google Translate / DeepL.

Ограничения

5–10 переводов в день бесплатно;

далее:

покупка за Stars;

или включено в Pro.

Кеширование

Все переводы сохраняются в БД, чтобы:

снижать расходы;

ускорять ответы.

6. Монетизация (встроена в продукт)
6.1 Freemium-модель (основа)
Бесплатно:

500 ⭐ — Pro на месяц

1 200 ⭐ — пожизненный доступ
